# Command Palette Navigation

Aperio provides a powerful global **Command Palette** to help you instantly search and navigate through large OpenAPI specifications. Instead of manually clicking through the sidebar to find an endpoint, you can search across all paths and tags in seconds.

## How to use the Command Palette

1.  Press the **Search** button in the top navigation bar, or press **`Ctrl+K`** (Windows/Linux) or **`Cmd+K`** (Mac) anywhere within the dashboard.
2.  Start typing the name of an endpoint, tag, or path you want to find.
3.  The Command Palette will instantly display categorized results for:
    *   **Resources**: High-level groupings based on your OpenAPI tags.
    *   **Endpoints**: Specific HTTP methods and paths (e.g., `GET /users`).
4.  Use your mouse or keyboard **up/down arrow keys** to highlight a result, and press **Enter** (or click) to navigate instantly to that endpoint.
