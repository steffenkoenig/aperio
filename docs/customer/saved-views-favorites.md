# Saved Views and Favorites

Aperio makes it easy to personalize your dashboard by pinning frequently used resources and saving table layouts. These settings are saved locally in your browser per API specification.

## Favorites (Pinning Resources)
If you interact with specific resources frequently, you can pin them to the top of your sidebar.
1. Hover over any resource in the sidebar.
2. Click the **Star icon** that appears on the right side of the item.
3. The resource will immediately appear in a dedicated "Favorites" section at the top of the sidebar.
4. Click the Star again to unpin it.

## Saved Views (Custom Tables)
If you frequently filter or sort data in resource tables, you can save those configurations as a "Saved View".
1. Navigate to any resource containing a data table (e.g., a GET collection endpoint).
2. Apply your desired filters using the search bar or sort by clicking column headers.
3. Click the **Save View** button next to the refresh button.
4. Enter a name (e.g., "Active Users") and save.
5. You can now use the dropdown menu next to the "Save View" button to quickly switch between your custom views and the "Default View".
6. To remove a view, select it from the dropdown and click the trash can icon.
